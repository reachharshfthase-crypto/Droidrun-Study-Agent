# Droidrun Study Agent

This project is built using the Droidrun Agent Framework and demonstrates agentic automation on Android apps.

## Description
The agent automates a study workflow by:
- Searching YouTube for an academic topic (Electromagnetism)
- Shortlisting and analyzing multiple videos
- Selecting the best educational video
- Orchestrating ChatGPT to explain the selected video
- Saving the video link and explanation as study notes

## Tech Stack
- Droidrun Agent Framework
- Mobilrun Cloud (Android)
- YouTube, ChatGPT, Notepad

## Agentic Behavior
The agent follows an observe–reason–act loop by autonomously selecting content and coordinating multiple Android apps.
